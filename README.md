Добрый день!
Проект релезован на Windows Forms C#, net 8.0
С испольованеи библиотеки MySql.Data

Для начала работы необходимо импортировать базу данных и изменить класс DB (файл dump.sql).
Использовалась MySQL в сборке OpenServer.

Программа имеет функцию авторизации (2 работника):
worker — работник
byxalter — бухгалтер
Пароль: 1

Так как ТЗ не запрещало и не описывало формат работы, она была реализована полностью для удалённого формата (никто не идёт пешком за справкой):

-Сотрудник создаёт запрос.
-Бухгалтер получает запрос.
-Бухгалтер берёт запрос в работу.
-Выполняет процесс создания справки и заносит её в электронный вид.
-Загружает справку в систему.
-Сотрудник получает и выгружает справку.

В ТЗ не указано, поэтому бухгалтеру не предоставлена возможность запроса справки.

Считаю, что программа полностью выполняет поставленные в ТЗ задачи.
